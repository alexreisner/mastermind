= PostgreSQL

== 'ident' Authentication Error

Use password instead of ident auth. Edit /etc/postgresql/8.3/main/pg_hba.conf and change the line at the bottom of the file that specifies 'ident':

  local   all         all                               password


== Dump/restore Databases

  pg_dump -U [user] [dbname] > [filename]
  psql    -U [user] [dbname] < [filename]
